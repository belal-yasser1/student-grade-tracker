# Student Grade Tracker 🎓

A Python project that tracks student grades using Pandas.

## Features
- Add students with their grades
- View pass/fail status per student
- Show statistics (average, min, max grade)
- Save and load data from CSV file

## Technologies Used
- Python
- Pandas

## How to Run
1. Install pandas: `pip install pandas`
2. Run the file: `python grade_tracker.py`
